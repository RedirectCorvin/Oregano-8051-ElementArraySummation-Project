# Oregano mc8051 ElementArraySummation Project
This github repo provides the initial C code, assembly code, binary instruction, and screenshot files to fulfill the requirements for Computer Organization course final project.

## Project Objective
The primary objective of this project is to implement and validate an 8-bit array summation program on the VHDL-based Oregano mc8051 IP Core. By simulating the execution in Mentor Graphics ModelSim, the project aims to analyze low-level hardware operations including Harvard memory mapping, instruction cycles, and ALU behavior and verify computational validity through waveform tracking.
